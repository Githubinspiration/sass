## Import

- `Sidebar`: The container component that manages state.
- `SidebarContainer`: The bare container component.
- `SidebarNav`: The sidebar navigation container.
- `SidebarNavGroup`: A group of navigation items.
- `SidebarLink`: The header component with navigation, title, toolbar and tabbar.
- `SidebarOverflow`: The scroll overflow.

```ts
import {
  Sidebar,
  SidebarContainer,
  SidebarNav,
  SidebarNavGroup,
  SidebarLink,
  SidebarOverflow,
} from '@saas-ui/pro'
```

## Usage

### Basic

```jsx
<Sidebar height="400px">
  <SidebarNav direction="row">
    <SaasUIGlyph width="24px" />
    <Spacer />
    <Menu>
      <MenuButton as={Button} variant="ghost">
        <PersonaAvatar presence="online" size="xs" src="/showcase-avatar.jpg" />
      </MenuButton>
      <MenuList>
        <MenuItem>Sign out</MenuItem>
      </MenuList>
    </Menu>
  </SidebarNav>
  <SidebarNav aria-label="Main">
    <SidebarLink label="Home" icon={<FiHome />} isActive />
    <SidebarLink label="Users" icon={<FiUsers />} />
    <SidebarLink label="Settings" icon={<FiSettings />} />
  </SidebarNav>
</Sidebar>
```

### Resizable sidebar

Use the `isResizable` prop to make the sidebar resizable. The default width is `280px`, min width `220px` and max width `320px`.

```jsx
  width="280px"
  minWidth="220px"
```

### Condensed variant

The condensed variant can be used as a collapsed state on smaller screens, or as the primary navigation in a double sidebar layout, see below.
SidebarLink labels will be rendered as tooltips.

```jsx
<Sidebar variant="condensed" height="400px">
  <SidebarNav>
    <SaasUIGlyph width="24px" />
  </SidebarNav>
  <SidebarNav>
    <SidebarLink label="Home" icon={<FiHome size="1.2em" />} size="md" />
    <SidebarLink label="Users" icon={<FiUsers size="1.2em" />} size="md" />
    <SidebarLink
      label="Settings"
      icon={<FiSettings size="1.2em" />}
      size="md"
    />
  </SidebarNav>
</Sidebar>
```

### Double sidebar

You can stack multiple sidebars together to create a double sidebar layout.

```jsx
<HStack>
  <Sidebar variant="condensed" height="400px" bg="purple.500" borderWidth="0">
    <SidebarNav>
      <SaasUIGlyph width="24px" variant="solid" color="white" />
    </SidebarNav>
    <SidebarNav>
      <SidebarLink
        label="Home"
        icon={<FiHome size="1.2em" color="white" />}
        size="md"
      />
      <SidebarLink
        label="Users"
        icon={<FiUsers size="1.2em" color="white" />}
        size="md"
        isActive
      />
      <SidebarLink
        label="Settings"
        icon={<FiSettings size="1.2em" color="white" />}
        size="md"

    </SidebarNav>
  </Sidebar>
  <Sidebar
    height="400px"
    isResizable
    width="280px"
    minWidth="220px"

    <SidebarNav direction="row">
      <Heading size="sm" fontWeight="semibold">
        Users

    <SidebarOverflow>
      <SidebarNav aria-label="Main">
        <SidebarNavGroup>

          <SidebarLink
            label="Customer"

              210
```
