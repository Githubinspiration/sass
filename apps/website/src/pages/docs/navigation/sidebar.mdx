## Import

- `Sidebar`: The container component that manages composition.
- `SidebarContainer`: The bare container component.
- `SidebarNav`: A button with an arrow icon used for navigating to the previous page.
- `SidebarNavGroup`: A button with an arrow icon used for navigating to the previous page.
- `SidebarLink`: The header component with navigation, title, toolbar and tabbar.
- `SidebarOverflow`: The page content.

```ts
import {
  Sidebar,
  SidebarContainer,
  SidebarNav,
  SidebarNavGroup,
  SidebarLink,
  SidebarOverflow,
} from '@saas-ui/pro'
```

## Usage

### Basic

```jsx
<Sidebar height="200px" />
```
