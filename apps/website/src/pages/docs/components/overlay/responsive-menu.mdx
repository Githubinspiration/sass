## Import

```ts
import { ResponsiveMenu, ResponsiveMenuList } from '@saas-ui/pro'
```

## Usage

ResponsiveMenu renders a `Menu` by default and switches to a `MenuDialog` on smaller screens.

```jsx
<ResponsiveMenu>
  <MenuButton as={Button}>Open menu</MenuButton>
  <ResponsiveMenuList>
    <MenuItem>Edit</MenuItem>
    <MenuItem>Delete</MenuItem>
  </ResponsiveMenuList>
</ResponsiveMenu>
```
