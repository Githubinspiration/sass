`Br` wraps a regular br tag and supports responsive styles,
making it easy to conditionally show or hide a new line on different screen sizes.

<ComponentLinks
  github={{ package: 'react' }}
  npm={{ package: '@saas-ui/react' }}
/>

## Import

```js
import { Br } from '@saas-ui/react'
```

## Usage

Setting the display to none will hide the new line.

```jsx
<Heading>
  This is my awesome
  <Br d={{ sm: 'inline', lg: 'none' }} /> hero header
</Heading>
```
