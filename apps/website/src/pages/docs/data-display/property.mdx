A `Property` can be used to format different kinds of data throughout your app.
It can be used in cards, lists, etc.

<ComponentLinks
  github={{ package: 'saas-ui-property' }}
  npm={{ package: '@saas-ui/property' }}
/>

## Import

- `Property`: The wrapper component that handles default composition.
- `PropertyLabel`: The property label.
- `PropertyValue`: The property value

```ts
import { Property, PropertyLabel, PropertyValue } from '@saas-ui/react'
```

## Usage

### Basic Property

```jsx
<Property label="Name" value="Elliot Alderson" />
```

### Label width

Label text will be truncated if it doesnt fit.

```jsx
<Property label="Billing plan" value="Professional" labelWidth="80px" />
```

### Property with icon

```jsx
<Property label={<EmailIcon />} value="hello@saas-ui.dev" labelWidth="32px" />
```

### Property list

```jsx
<List>
  <Property label="Name" value="Elliot Alderson" />
</List>
```

### Editable property

```jsx
<Stack spacing="2">
  <Property
    label="Name"
    value={
      <Editable defaultValue="Eelco">
        <EditablePreview />
        <EditableInput />
      </Editable>
    }
  />
  <Property
    label="Status"
    value={
      <Select value="Open">
        <MenuItemOption value="Open">Open</MenuItemOption>
        <MenuItemOption value="Closed">Closed</MenuItemOption>
      </Select>
    }
  />
</Stack>
```
