title: LoadingOverlay
description: A loading indicator that overlays it's parent.
---

The `LoadingOverlay` component will overlay all content of it's parent element and
renders a `Spinner` in the center.

  }}
  npm={{ package: '@saas-ui/layout' }}
/>

## Import

```ts
import { LoadingOverlay } from '@saas-ui/react'
```

## Usage

Be sure to add `position="relative"` to the parent div, so the component overlays correctly.

### Basic

```jsx
<Card height="100px" title="Users" position="relative">
  <LoadingOverlay />
</Card>
```
