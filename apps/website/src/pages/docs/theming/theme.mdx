The theme object is where you define your application's color palette, type
scale, font stacks, breakpoints, border radius values, and more.

All Saas UI components come with a default style config that compliments
the default Chakra UI theme.

## Setup your theme

To make sure all Saas UI components are styled correctly, you will need to extend
your theme, using the Saas UI base theme.

```jsx live=false
// 1. Import the base theme
import { extendTheme } from '@chakra-ui/react'
import { baseTheme } from '@saas-ui/react'

// 2. Add the base theme to your theme configuration
const theme = extendTheme({
  // ... your custom theme
}, baseTheme)

// 3. Pass the theme to ChakraProvider
<ChakraProvider theme={theme}>
  <App />
</ChakraProvider>

```

More information on customizing your theme [can be found here.](https://chakra-ui.com/docs/theming/customize-theme)
