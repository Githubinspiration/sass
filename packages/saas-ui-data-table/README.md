# @saas-ui/data-table

A DataTable component for Chakra UI, build with react-table.

## Installation

```sh
$ yarn add @saas-ui/data-table

#or

$ npm i @saas-ui/data-table  --save
```

## Usage

```jsx
<DataTable data={data} columns={columns} />
```
