# @saas-ui/layout

Additional layout components for Chakra UI.

- EmptyState
- Divider (with label support)
- Loader (Loading overlay)

## Installation

```sh
$ yarn add @saas-ui/layout

#or

$ npm i @saas-ui/layout  --save
```
