# @saas-ui/forms

Chakra UI Forms.

Create fully functional React forms with just a few lines of code.

Uses React Hook Form under the hood.

## Installation

```sh
$ yarn add @saas-ui/forms

#or

$ npm i @saas-ui/forms  --save
```
