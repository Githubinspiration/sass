# @saas-ui/input-right-button

Input right button component for Chakra UI.

## Installation

```sh
$ yarn add @saas-ui/input-right-button

#or

$ npm i @saas-ui/input-right-button --save
```

## Usage

```ts
<InputGroup>
  <Input />
  <InputRightButton>
    <Icon />
  </InputRightButton>
</InputGroup>
```
